I want to make webpages

So you want to make pretty webpages and stuff? All webpages have a basic backbone of 3 coding languages: HTML, CSS and Javascript. This includes the simplest blog layout to massive, shiny corporate websites to your favourite ecommerce site such as Amazon. But don't worry if 3 languages sounds like a lot, there are plenty of free resources online and "in real life" which can help you get started. Click on each of the buttons below to find out more about each language or on "I want to start now!" to see some great resources that we've curated here at #Website_name.
