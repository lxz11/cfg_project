Where do I start?

We'll go straight to the nitty gritty here: there are lots of places to start learning programming but it all depends on what you want to do. For example, constructing and styling websites is a excellent entry point as it is possible to create some great looking and practical webpages in a relatively short amount of time. On the other hand, creating software, such as games, requires somewhat more in depth knowledge of a programming language and may take longer before practical results can be seen. In all cases, it takes dedication and practice to become truly good but we all have to start somewhere!

Click the links below to skip to the relevant section or scroll down to read a little more about each area. We couldn't possibly cover all the things you can do with programming though so make sure to search around and read up on your chosen area.

I want to make webpages

Making websites is a very rewarding experience where you can see all the fruits of your labour right there in your browser at any time. The start, All webpages have a basic backbone of 3 coding languages: HTML, CSS and Javascript. This includes the simplest blog layout to massive, shiny corporate websites to your favourite ecommerce site. But don't worry if 3 languages sounds like a lot, there are plenty of free resources online and "in real life" which can help you get started. Click on the buttons below to find out more about each language or on "I want to start now!" to skip to the resources that will help you learn.

HTML stands for Hypertext Markup Language.

This means that it provides the structure and layout of the website which is done so using tags. Without an html structure, there is no website! The newest iteration of HTML is HTML5 which has some small differences from HTML4 but also some really cool new features that you can mess around with. It's recommended to learn the newest version as older versions are gradually phased out and may be at risk of security issues. That said, HTML5 and browsers in general are all backwards compatible so an old website written in HTML2 would still display the same as it did when it was first written.

CSS stands for Cascading Stylesheets

You always need to first have an HTML structure down for your webpage but it will still look very plain. That's where CSS comes in. CSS determines all the styling on the page including the colours, shapes, fonts and sizes of elements. It does so by referencing an element that was established in the HTML and changing an attribute associated with it. The CSS of a page may be written in the same document as the HTML but common practice is to write it in a separate file and to link it to the main HTML document. In this way, you can completely change the look of a website despite it having the same HTML structure.

Javascript stands for... well, just Javascript

Lets go with an analogy here. If HTML is the tabletop and legs of a table, then CSS is the paint or carvings on the table to make it look nice. Javascript would then be adding drawers and additional functionality to the table that make your experience more enjoyable.

That's right, Javascript provides interactivity to a website such as pop-up boxes, animations and much much more. But Javascript is also a very broad language and there are 'libraries' which people have developed to make doing some things much easier and one of most common of these is jQuery which most of our recommended resources choose to teach.

I want to start now!

I want to make software applications

Making usable software requires understanding an Object Oriented Language.

Python is named after Monty Python.

Of the three languages mentioned in this section, Python is the youngest and is still gaining popularity. It's often recommended for beginners due to being easy to read and learn and there's a plethora of resources and documentation in the form of online material as well as books to help you get started. A technical caveat of Python compared to other languages is that it runs slower due to being hugely flexible.

Java is not related to Javascript!

Java is a staple in most Computer Science degrees as it was designed to be more beginner friendly than C++ by handling more complex functions for you, such as memory management. It is also hugely popular within industry, thus has a lot of resources and documentation available too AND it would be a valuable skill to have for career options. One downside to Java is though, is that it is very wordy and requires a lot of coding to reach a usable application.

C++ is "one better than C" (ie. it's a follow on of the language C)

C++ is the oldest of the languages mentioned here and is deemed as the most difficult to learn. However but is arguably the most powerful. If you're looking to build complex programs, systems or games, then C++ is the way to go, just be prepared to put in the long hours to truly master this language.

Resources to use

So now that you know a little bit about the languages to learn, you're raring to go. But just a moment, there are many resources that teach the same or similar topics in different styles, some are free, some require a one-off payment and some are subscription services. We recommend trying out some free trials before settling for a particular course to see if the teaching style suits you.

Codecademy

Price: Free for courses, subscription for extra projects and features.

Languages: HTML&CSS, Javascript, Java, Python, Ruby, SQL, PHP

Pros: Many courses for different languages! Has an interactive learning environment. Great for learning the basics!

Cons: Limited hints which may result in being stuck for a while.

General Assembly Dash

Price: Some free starter courses, subscription for full course.

Languages: HTML&CSS, Javascript, other courses relating to design

Pros: Great interface and pacing of tutorials with interactive learning environment.

Cons: Limited free offerings so only covers the very basics of HTML, CSS and Javascript.

Freecodecamp

Price: Completely free!

Languages: HTML&CSS, Javascript, other front-end and back-end skills.

Pros: Project based learning and a chance to work on projects for not-for-profit organisations.

Cons: Requires a lot of researching and self-study.

Treehouse

Price: Free 7-day trial then subscription

Languages: HTML&CSS, Javascript, Java, Python, Ruby, PHP, C#, databases and others

Pros: Very guided course with lots of reinforcement. Recommended for beginners.

Cons: Trial period is only 7 days.

Udacity

Price: Free introductory courses. Subscription service for others but get an extra month free trial with Github Student Developer Pack.

Languages: HTML&CSS, Javascript, other courses relating to design

Pros: Nanodegree courses are created in conjunction with market leading companies eg. Amazon, Google.

Cons: On the pricier side although there is a discount incentive to complete the course within 12 months.

Lynda

Price: Free 10-day trial before subscription.

Languages: Many courses available for different IT applications. For example, animation, web, design etc.

Pros: Higher education students may already have a subscription with them, thus making it free! Subscription covers a huge variety of courses.

Cons: Primarily video instruction so is less interactive than the alternatives.
